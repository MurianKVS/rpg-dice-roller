# 🎲 RPG Dice Roller

🔗 **Demo:** https://murianKVS.github.io/rpg-dice-roller

Um simulador de dados para RPG de mesa desenvolvido em **HTML, CSS e JavaScript**.

O projeto permite rolar diferentes tipos de dados utilizados em sistemas de RPG, exibindo o resultado com animação, histórico de rolagens e detecção de críticos.

---

## 🚀 Funcionalidades

- 🎲 Rolagem de múltiplos dados:  
  `d2`, `d4`, `d6`, `d8`, `d10`, `d12`, `d20`, `d100`

- ⚡ Animação de rolagem ao clicar no botão

- 📜 Histórico das **últimas 10 rolagens**

- 🔥 Detecção de **Crítico (20)** no d20

- 💀 Detecção de **Falha Crítica (1)** no d20

- 🎨 Destaque visual para:
  - última rolagem
  - críticos
  - falhas críticas

- 📱 Interface parcialmente responsiva

---

## 🛠 Tecnologias utilizadas

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**

---

## 🧠 O que foi praticado neste projeto

- Manipulação do DOM
- Eventos em JavaScript
- Geração de números aleatórios
- Manipulação dinâmica de elementos HTML
- Controle de estado simples (histórico de rolagens)
- Estruturação de lógica em funções reutilizáveis
